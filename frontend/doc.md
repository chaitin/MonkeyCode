# MonkeyCode AI 前端使用文档

本文档基于当前 `frontend` 源码整理，描述的是现在前端已经实现的页面、入口和交互，不再沿用旧版文档中已经下线或改版的说明。

---

## 一、产品简介

MonkeyCode 是一套面向研发团队的 AI 开发平台，当前前端已经覆盖以下几类核心能力：

- 首页与公开介绍页
- 普通用户控制台
- AI 智能任务创建与协作开发
- 项目与需求管理
- Git 机器人 / 自动 Review
- 独立开发环境、终端、文件管理、远程协助
- 开发者广场内容浏览与发布
- 企业管理后台

### 主要入口

- 首页：`/`
- 登录页：`/login`
- 开发者广场：`/playground`
- 发布广场内容：`/playground/create`
- 广场详情：`/playground/detail?id=...`
- 普通用户控制台：`/console`
- 新任务页：`/console/tasks`
- 任务详情页：`/console/task/:taskId`
- 项目页：`/console/project/:projectId`
- Git 机器人：`/console/gitbot`
- IDE 辅助工具：`/console/ide`
- 独立终端页：`/console/terminal?envid=...`
- 独立文件管理页：`/console/files?envid=...&path=...`
- 远程协助页：`/sharedterminal?id=...`
- 企业管理后台：`/manager`

### 首页导航

首页顶部导航当前包含：

- `产品介绍`
- `广场`
- `使用文档`
- 已登录时显示 `控制台`
- 未登录时显示 `注册`、`登录`

移动端会收纳为抽屉菜单。

### 首页内容

首页目前由以下区块组成：

- Banner / 产品介绍
- 能力亮点
- 全平台客户端下载
- 任务协作展示
- Git 机器人介绍
- 套餐与价格
- 页脚资源链接与技术交流群二维码

### 客户端下载

首页“全平台客户端”区块会跳转到 GitHub Releases，当前展示的平台有：

- Windows
- macOS
- Android
- iOS

下载地址：

- `https://github.com/chaitin/MonkeyCode/releases`

## 二、登录与账号

### 登录页

登录页路径：`/login`

页面分为两个标签：

- `普通用户`
- `团队管理员`

两种登录表单都支持：

- 输入邮箱与密码
- 密码明文/隐藏切换
- 登录前验证码校验
- 浏览器本地记住上次输入过的邮箱和密码

### 普通用户登录

普通用户登录成功后进入：

- `/console/`

普通用户标签下还提供：

- `找回密码`
- `百智云账号登录`
- `注册`

其中“百智云账号登录”和“注册”都通过后端登录入口发起。

### 团队管理员登录

团队管理员登录成功后进入：

- `/manager/`

团队管理员标签下当前没有单独的“找回密码”按钮。

### 找回密码

找回密码页路径：

- `/findpassword`

操作流程：

1. 输入邮箱
2. 完成验证码校验
3. 提交后发送重置密码邮件

### 重置密码

重置密码页路径：

- `/resetpassword?token=...`

页面会先根据 `token` 读取账号信息，再要求输入：

- 新密码
- 确认密码

重置成功后弹出确认框，关闭后返回首页。

## 三、普通用户控制台

登录后默认进入：

- `/console/tasks`

当前控制台布局为：

- 左侧侧边栏
- 顶部面包屑与刷新区
- 主内容区域

### 左侧侧边栏

侧边栏分为三块：

1. `开发项目`
2. `配置`
3. 钱包与当前用户

#### 开发项目

“开发项目”区域包含：

- `默认`
- 用户创建的项目列表

支持的交互：

- 刷新项目与任务列表
- 创建项目
- 展开/折叠项目
- 自动记住展开状态
- 任务执行中时自动定时刷新

`默认` 分组对应未归属项目的任务，点击进入：

- `/console/tasks`

每个任务项支持：

- 进入任务详情
- 执行中任务可 `终止任务`
- 所有任务可 `删除任务`

每个项目项支持：

- 进入项目概览页
- 项目已绑定仓库时可直接 `启动任务`
- 查看项目下已有任务

### 顶部区域

个人控制台顶部当前包含：

- 面包屑导航
- `技术交流群`
- `刷新`

“技术交流群”悬浮后会显示：

- 微信群二维码
- 飞书群二维码
- 钉钉群二维码

### 钱包

钱包入口位于侧边栏底部，不在顶栏。

钱包弹窗包含四个分区：

- `积分余额`
- `我的套餐`
- `赚积分`
- `积分记录`

当前已实现的能力包括：

- 查看账户积分与每日积分
- 选择积分充值档位
- 开通专业版
- 专业版自动续费开关
- 使用兑换码兑换积分
- 复制邀请链接
- 查看邀请记录
- 查看积分收支流水

### 用户菜单

侧边栏最下方的用户菜单支持：

- 修改头像
- 修改昵称
- 绑定邮箱
- 修改密码 / 设置密码
- 登出

说明：

- 头像文件必须为图片，且不超过 5MB
- 已绑定邮箱时，“绑定邮箱”按钮会禁用
- 已有密码的账号修改密码时需要输入当前密码

## 四、智能任务

### 新任务页

新任务页路径：

- `/console/tasks`

页面由两部分组成：

- 上方任务输入区
- 下方任务卡片列表

### 创建任务

创建任务时可配置以下内容。

#### 1. 输入任务内容

主输入框支持直接输入自然语言需求。

#### 2. 选择任务类型

当前有三种类型：

- `开发`
- `设计`
- `审查`

#### 3. 选择代码来源

“代码”菜单支持三种来源：

- `ZIP 文件`
- `我的仓库`
- `其他仓库`

`我的仓库`：

- 会按已绑定 Git 身份分组展示
- 支持在每个身份下搜索仓库
- 选中后自动关联对应身份凭证

`其他仓库`：

- 直接输入完整仓库地址
- 按回车确认
- 下方会显示历史仓库地址

`ZIP 文件`：

- 只接受 `.zip`
- 文件大小上限 10MB
- 上传成功后作为本次任务代码来源

#### 4. 选择技能

“技能”按钮会弹出技能选择面板，按标签分类展示。

说明：

- 支持多选
- 部分默认技能不可取消
- 按钮上会显示已选技能数量

#### 5. 语音输入

任务输入框支持语音转文字。

#### 6. 执行前参数确认

点击 `执行` 后会打开“任务参数”弹窗。

当前弹窗字段为：

- 大模型
- 仓库分支
- 仓库身份凭证

仅子账号额外显示：

- 宿主机
- 系统镜像

补充说明：

- 从“我的仓库”选中的仓库，不再重复要求选择身份凭证
- 如果使用 ZIP 代码源，则不会显示仓库分支和仓库凭证
- 当前创建的开发任务固定使用 `OpenCode`
- 如果达到任务并发上限，会弹出并发限制提示框

### 任务列表

任务列表以卡片方式展示，当前每页加载 24 条，支持无限滚动。

卡片信息包括：

- 任务标题
- 仓库名或 ZIP 文件名
- 任务状态
- Token 用量
- 创建时间

任务状态当前对应：

- `等待执行`
- `正在执行`
- `任务完成`
- `执行失败`

卡片右上角菜单支持：

- 执行中的任务 `终止`
- 删除任务

## 五、任务详情页

任务详情页路径：

- `/console/task/:taskId`

这是新版文档里最需要更新的部分。当前实现与旧版“底部文件/终端/修改/续期工具条”已经不同。

### 页面结构

当前任务详情页主要由以下区域组成：

- 顶部简要状态栏
- 左侧消息区
- 右侧文件区（按需展开）
- 底部终端区（按需展开）
- 预览弹窗

顶部按钮当前只有：

- `终端`
- `文件`
- `预览`

说明：

- `文件` 按钮会显示当前已变更文件数量
- `预览` 按钮会显示可用预览端口数量
- 当开发环境不可用时，这三个按钮会禁用

### 环境准备状态

如果任务对应开发环境还未就绪，主区域会显示准备视图，例如：

- 正在创建虚拟机
- 正在准备环境

### 消息区

消息区会展示任务往返消息与工具执行信息，支持：

- 历史消息懒加载
- 自动滚动到底部
- 一键滚动到顶部 / 底部

任务结束后，如果当前轮次消息尚未附着在流式连接上，页面会自动加载历史轮次消息。

### 与 AI 对话

输入区支持：

- `Enter` 发送
- `Shift + Enter` 换行
- 语音输入
- 斜杠命令插入

输入框左侧菜单当前有两项：

- `重置上下文`
- `重新加载开发工具`

任务执行中时：

- 输入框会隐藏
- 右侧发送按钮变成 `停止` 按钮
- 会显示当前耗时

任务已结束时：

- 输入区会显示 `任务已结束`

### 文件面板

点击顶部 `文件` 后，会在右侧展开文件面板。

文件面板当前包含两个模式：

- `目录`
- `变更`

#### 目录模式

支持：

- 浏览任务工作区文件树
- 打开文本文件预览
- 二进制文件提示不可预览
- 大文件提示不可预览
- 刷新目录
- 在目录中创建文件
- 创建目录
- 上传文件
- 复制文件
- 移动文件
- 下载文件/目录
- 删除文件/目录

#### 变更模式

支持：

- 查看所有已变更文件列表
- 按文件查看新增/删除行数
- 打开单个文件的 Diff
- 分栏显示代码差异

说明：

- 旧文档里的独立“修改”面板已经没有了，变更查看功能被合并进右侧文件面板

### 终端面板

点击顶部 `终端` 后，会在页面下方展开终端区。

终端区用于查看和操作当前任务对应的终端连接。

### 在线预览

点击顶部 `预览` 会打开预览弹窗。

弹窗当前行为：

- 列出当前开发环境中已发现的监听端口
- 如果端口已有可访问地址，会显示 `访问`
- 如果端口不可访问，会显示错误或“暂不可访问”

说明：

- 任务详情页里的预览只负责展示与访问
- 端口的开放、关闭、白名单编辑在独立终端页的“在线预览 / 端口管理”里完成

## 六、Git 机器人 / 代码审查

Git 机器人页面路径：

- `/console/gitbot`

当前没有固定的侧边栏入口，需要直接访问地址。

页面包含两个标签：

- `审查任务`
- `机器人`

### 审查任务

支持：

- 查看机器人触发的审查任务列表
- 显示关联平台与状态
- 点击后跳转到对应 Git 平台页面
- 手动刷新列表

### 机器人

支持：

- 创建审查机器人
- 编辑机器人
- 修改机器人权限

创建机器人时需要填写：

- 备注
- Git 平台
- Access Token
- 宿主机

创建成功后会展示：

- Webhook 地址
- Secret Token

供你回填到对应 Git 平台。

## 七、项目协作

项目页路径：

- `/console/project/:projectId`

### 项目头部

项目头部区域会展示：

- 项目名称
- 项目描述或“未绑定仓库”提示
- 自动 Review 状态
- `启动 AI`
- 更多菜单

更多菜单当前包括：

- 修改名称
- 环境变量
- 开发镜像
- 移除项目

### 自动 Review

点击 `未开启自动 Review / 已开启自动 Review` 可进入配置弹窗。

开启后：

- 前端会调用项目自动 Review 接口
- 文案说明为：自动配置仓库 Webhook
- 当有新的 PR / MR 时，MonkeyCode 会自动启动 Review 任务

### 启动 AI

项目页的“启动 AI”不是回到通用新任务页，而是弹出一个针对当前项目的对话框。

当前可配置：

- 项目
- 仓库分支
- 大模型
- 任务内容

提交后会直接创建一个与项目关联的开发任务。

### 项目页页签

当前项目页固定有三个页签：

- `信息`
- `需求`
- `任务`

#### 信息

“信息”页签当前包含两块内容：

- 项目文件管理器
- 项目根目录 `README.md` 自动加载与渲染

如果仓库里没有 `README.md`，会显示“暂无文档”。

#### 需求

“需求”页签支持：

- 按状态筛选
- 按优先级筛选
- 创建需求
- 查看需求详情

需求详情弹窗当前支持：

- 直接编辑标题
- 编辑原始需求文档
- 编辑技术方案文档
- 修改状态
- 修改优先级
- 查看创建者与创建时间

需求的“更多操作”中支持：

- 启动设计任务
- 启动开发任务

#### 任务

“任务”页签展示当前项目下的任务卡片列表，支持：

- 查看任务详情
- 无限滚动加载
- 删除任务

## 八、配置

配置通过个人控制台左侧侧边栏底部 `配置` 打开。

当前配置弹窗固定有 6 个分区：

- `Git 身份`
- `AI 大模型`
- `系统镜像`
- `宿主机`
- `开发环境`
- `通知`

### 1. Git 身份

当前支持的绑定方式包括：

- GitHub App 安装
- Gitee 授权绑定
- Gitea 授权绑定
- GitLab 授权绑定
- 对部分平台手动录入 `Base URL / Token / Username / Email / 备注`

已绑定身份支持：

- 查看平台信息
- 修改
- 移除

### 2. AI 大模型

模型列表支持：

- 查看模型健康状态
- 查看接口类型
- 查看归属类型（平台内置 / 私有）
- 设为默认
- 健康检查
- 修改私有模型
- 移除私有模型

### 3. 系统镜像

镜像列表支持：

- 查看镜像备注和镜像名
- 查看归属类型
- 设为默认
- 修改私有镜像
- 移除镜像

### 4. 宿主机

宿主机页面支持：

- 生成并复制安装命令
- 通过命令绑定 Linux 宿主机
- 查看宿主机状态与使用中的开发环境数量
- 设为默认
- 修改备注
- 移除私有宿主机

### 5. 开发环境

这里指独立创建和管理的开发环境，不是任务详情页里的临时交互区。

支持：

- 创建开发环境
- 查看在线 / 离线环境
- 打开该环境的终端
- 打开该环境的文件管理
- 续期
- 移除

创建开发环境时可配置：

- 宿主机
- 开发环境名称
- 仓库地址
- 仓库分支
- 仓库身份凭证
- 系统镜像
- AI 大模型
- CPU
- 内存
- 生命周期

补充说明：

- 平台内置宿主机会限制可选生命周期上限
- 页面支持切换是否显示离线开发环境

### 6. 通知

用户级通知当前支持四类接收端：

- 钉钉机器人
- 飞书机器人
- 企业微信机器人
- Webhook

每个接收端支持：

- 名称
- Webhook 地址
- Secret（可选）
- 订阅事件类型

可执行操作：

- 添加
- 编辑
- 测试发送
- 移除

## 九、独立终端、文件管理与远程协助

### 独立终端

路径：

- `/console/terminal?envid=...`

这是针对某个开发环境单独打开的终端页。

当前能力包括：

- 查看开发环境名称、状态、操作系统
- 选择已有终端连接
- 新建终端连接
- 关闭终端连接
- 重新连接
- 切换终端配色主题
- 打开文件管理
- 打开端口管理 / 在线预览
- 发起远程协助

### 端口管理

独立终端页的“在线预览”实际打开的是端口管理弹窗，支持：

- 查看端口列表
- 开放访问
- 访问已开放端口
- 复制访问地址
- 编辑白名单
- 关闭访问

开放访问时，前端会先尝试获取当前访问者 IP，并将其写入白名单。

### 远程协助

独立终端页支持生成远程协助连接信息。

模式有两种：

- `只读模式`
- `控制模式`

生成后会得到：

- 共享地址
- 连接密码

支持一键复制整段邀请信息。

### 远程协助页

路径：

- `/sharedterminal?id=...`

访问者进入后需要先输入连接密码，之后可加入共享终端。

页面支持：

- 查看分享者信息
- 查看连接状态
- 切换终端主题

### 独立文件管理

路径：

- `/console/files?envid=...&path=...`

这是针对某个开发环境的完整文件管理页。

支持：

- 浏览目录
- 面包屑导航
- 新建文件
- 新建目录
- 上传文件
- 复制
- 移动
- 下载
- 删除
- 在线编辑文本文件

说明：

- 超过 1MB 的文件不允许在线编辑

## 十、IDE 辅助工具

路径：

- `/console/ide`

## 十一、开发者广场

### 广场列表

路径：

- `/playground`

当前已实现：

- 搜索内容
- 浏览文章 / 任务分享卡片
- 查看作者与浏览量
- 登录后发布内容
- 未登录点击“发布”会跳到登录页

### 发布内容

路径：

- `/playground/create`

当前可选发布类型只有两种：

- `写一篇文章`
- `分享你执行过的任务`

旧文档里如果提到“项目分享”等功能，应以源码为准，目前前端没有完整开放这一类型。

#### 文章发布

支持：

- 标题
- Markdown 内容
- 图片上传

#### 任务分享

支持：

- 选择自己已有的任务
- 填写标题
- 填写 Markdown 内容
- 上传封面图片

补充说明：

- 当前前端要求至少上传 1 张图片才能发布
- 如果所选任务仍处于执行中，且开发环境在线，还可以额外选择“产出物”文件
- 这些产出物会被前端临时打包为 ZIP 后一起发布

### 广场详情

路径：

- `/playground/detail?id=...`

广场详情页会根据内容类型展示：

- 标题、作者、发布时间
- Markdown 正文
- 图片预览

对于任务分享，还额外支持：

- 播放任务消息回放
- 查看分享时附带的 ZIP 代码产出物
- 文件树展开 / 折叠
- 打开单个文件内容预览

## 十二、公开任务

路径：

- `/tasks/public?id=...`

这个页面当前不是独立展示页，而是一个跳转入口：

- 如果带有 `id`，会直接跳转到 `/console/task/:taskId`
- 如果没有 `id`，会跳回首页

所以旧文档里如果把它描述成单独的“公开任务列表页”，已经不准确。

## 十三、企业管理后台

企业管理后台路径：

- `/manager`

默认进入：

- `/manager/dashboard`

左侧菜单当前包含：

- 仪表盘
- 成员管理
- 开发环境
- AI 大模型
- 系统镜像
- 管理员
- 操作记录
- 其他配置

### 1. 仪表盘

当前仍是占位页，显示“敬请期待”。

### 2. 成员管理

成员管理页由两张卡片组成：

- 团队成员
- 分组

#### 团队成员

支持：

- 查看成员数量与上限
- 批量添加成员
- 添加时一次输入多个邮箱
- 禁用 / 启用成员
- 为成员发送重置密码邮件
- 查看成员加入时间和最近使用时间
- 查看成员所属分组

#### 分组

支持：

- 新建分组
- 修改分组名称
- 删除分组
- 查看分组成员
- 调整分组成员

### 3. 开发环境

团队级开发环境页面支持：

- 绑定宿主机
- 查看宿主机状态、规格、IP
- 修改宿主机信息
- 删除宿主机
- 查看宿主机所属分组
- 查看宿主机上正在使用的虚拟机

### 4. AI 大模型

团队模型支持：

- 添加
- 修改
- 健康检查
- 删除
- 查看所属分组

### 5. 系统镜像

团队镜像支持：

- 添加
- 修改
- 删除
- 查看所属分组

### 6. 管理员

管理员页支持：

- 查看所有管理员
- 添加管理员
- 为管理员发送密码重置邮件

### 7. 操作记录

操作记录页支持：

- 查看审计日志表格
- 查看时间、用户、操作、来源 IP
- 查看请求内容
- 查看响应内容
- 游标分页
- 切换每页条数

### 8. 其他配置

当前“其他配置”里已经实现的是团队级消息通知。

能力与用户侧通知基本一致，支持：

- 钉钉机器人
- 飞书机器人
- 企业微信机器人
- Webhook
- 添加 / 编辑 / 测试 / 删除
- 选择订阅事件

## 十四、和旧文档相比的主要变化

这里列出几条最容易误导的旧描述，便于你快速校对。

- 个人控制台的钱包入口现在在侧边栏底部，不在顶栏。
- 任务详情页现在是顶部 `终端 / 文件 / 预览` 按钮，不再是旧版底部工具条。
- 任务详情页没有独立“修改”面板，变更查看已经合并到右侧文件面板里的 `变更` 模式。
- 任务详情页没有旧文档里描述的“续期”按钮；开发环境续期在独立开发环境管理页中完成。
- 项目页现在固定是 `信息 / 需求 / 任务` 三个页签。
- 项目信息页现在包含项目文件管理器和 README 自动渲染。
- IDE 辅助工具页仍然是占位页，不是已完整交付的功能。
- 企业管理后台的仪表盘目前仍是占位页。
- `/tasks/public` 现在只是跳转页，不是独立公开列表页。
